# MyIP
====
Plasma 5 Widget to display your IP and ISP

## Install
### Clone This Project
```
git clone https://github.com/biofects/MyIP-Plasmoid.git
```
### Change Directory 
```
cd MyIP-Plasmoid
```
### Install Plasmoid
```
plasmapkg2 -i MyIP
```

### Add to your Desktop
```
1. Right click on desktop
2. Add Widgets
3. Search MyIp
4. Add and resize if needed
```
